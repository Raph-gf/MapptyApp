📚 Contexte Pédagogique
Ce projet fait partie du cours "The Complete JavaScript Course 2024" de Jonas Schmedtmann sur Udemy. Il a été développé dans le but de :

Maîtriser les concepts avancés de Programmation Orientée Objet (OOP) en JavaScript

Comprendre l'architecture des applications modernes

Manipuler des APIs comme Leaflet et Geolocation

🎯 Objectifs d'Apprentissage
OOP en JavaScript :

Classes et héritage (ES6)

Encapsulation avec les champs privés (#)

Polymorphisme

Architecture :

Pattern MVC (Modèle-Vue-Contrôleur)

Séparation des responsabilités

APIs Externes :

Intégration de Leaflet.js

Utilisation de la Geolocation API

🙏 Remerciements
Ce projet a été réalisé dans le cadre du cours exceptionnel de Jonas Schmedtmann. Les concepts enseignés ont permis de :

Comprendre l'OOP moderne en JS

Maîtriser les APIs navigateur

Structurer des applications complexes

Note pédagogique : Ce code est destiné à des fins d'apprentissage. Pour une utilisation en production, une refactorisation serait nécessaire.
